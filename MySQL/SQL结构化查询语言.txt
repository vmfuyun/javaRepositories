表的概念  

什么是表(table)

 表中的列:           列/字段Field 
 表中的行:	          一条数据/一条记录

面向对象的思想  

表                       类               定义存储的结构  
列	          字段            定义存储那些属性
行 	           对象           表示一条数据/一个对象





数据库的操作和存储引擎
主要有 表 , 视图,存储过程,函数,触发器,事件,等      
数据库: 存储 数据库对象的容器  

常用的命令  
查看 有哪些数据库    
SHOW   DATA BASES;

使用指定的数据库;
USE database_name;

查看有哪些数据表
SHOW TABLES;

创建指定名称的数据库 
create database  database_name;

删除数据库
drop database   database_name;




注解的定义  
@Target(ElementType.TYPE)      //表示能贴在哪   注意是贴在类还是 贴在 哪
@Retention(RetentionPolicy.RUNTIME)     //表示能 保存到哪个时期 
			 例如 runtime时期 就是运行的时候 也会起作用  一般自定义一定要是这个
			        class 时期 是不会出现在运行的时候
			     还有就是源码  sourec  
public  @interface 注解名称  {
	//抽象方法
	// 属性(在注解中 抽象方法 叫属性)
	//返回类型  方法名() default  值               
	}
  接口一般没有 字段(属性)   大多 有方法    就是     有没有   返回值


这个第三方的 赋能 就是通过 得到 是否 贴上了标签 (通过class反射来赋予功能  来获取比如getAnnotations  ) 然后进行判断运行


使用语法    
	@注解名(属性1=属性值1,属性名2=属性值2......)

对于结构性的  东西 就要找字节码对象

通过字节码对象可以得到 注解 的对象  
 通过对象就可以 注解的字段 啦什么方法啦   之类


注意    它的第三方功能 能办很多事

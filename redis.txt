list类型

	lrange key start end 查看--指定范围 全部的 输入 0  -1
	linsert  key before|after refval  newVal

set类型
	sadd key value  添加set集合
	sinter 求交集  
	sinterstore var key1 key2  把key1和key2的交集 存入 var
	sunion key1 key2 并集   
	srandmember key count 随机获取set集合中的元素
	smembers key 查看
	sdiff 比较差异 输出左边的集合的
	
zset类型
	每一元素都会有关联一个double分数  用来排序
	zadd key num member    --num用来排序 
	zrange key start end  排序输出
	zrangebyscore key min max  (可以带有分值输出)指定分数范围来排序
	zrank key member   指定member 来看是第几个索引
	zremrangebyscore key min max 通过分数删除(也有通过排名删除)
	zcount

应用场景
	经常查询的 数据  存在Redis中比较合适(当缓存)  一般Redis存的value是 json
	Redis在一段时间内 会把数据存在磁盘中 incr 递增加1 (点赞)
	热门推荐 把首页的数据以list形式存在Redis
	 抽奖 set形式
	好友推荐  有可能认识的人 -- 用的 是set的集合 操作   (有特殊符号的 使用"" 比较好)    
			先把1 对应的好友的 集合 求并集
			然后 并集把1 去掉  
			然后再随机推送

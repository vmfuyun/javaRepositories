
     package  是包 类似于文件夹

 语法格式   package 包名.子包名.子子包;   必须把该语句作为java文件中,并且是            第一行代码  
  


 编译命令变成

   javac -d . java文件名.java     
   
   -d  为当前目录  如果没有使用Package的语句 会编译在当前文件 
                   如果有使用的话     会把编译后的字节码放在 对应名字的包里


运行的话   使用 java 包名(或者是子包名).名字  运行  
        注意子包和主包直接要用 . 来分割
